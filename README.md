
# Mentor

# Requirements

1. Pyenv
2. Poetry
3. Docker

# Installation

1. Clone the repository:
   ```bash
   git clone git@github.com:carolrizzi/mentor.git
   cd mentor
   ```
2. Install dependencies:
   ```bash
   poetry install
   ```
3. Run docker compose:
   ```bash
   cd deploy
   docker compose up -d
   ```
